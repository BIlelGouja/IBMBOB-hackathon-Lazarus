## Executive Summary
- Lazarus does not call IBM Bob from the browser. The frontend sends code to the internal Next.js route `/api/bob`.
- The backend route reads `process.env.IBM_BOB_API_KEY`, builds a chat-completions request, and sends the serialized workspace to IBM Bob.
- The Deep Context task proves that IBM Bob receives repository-like context, not only a single isolated snippet.
- The report is safe to publish because it documents environment variable names without exposing real secret values.
- The returned structure powers the later Lazarus map, security audit, live recode, diff viewer, and downloadable artifacts.
